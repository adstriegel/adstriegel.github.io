CSE 30341 - Operating System Principles - Spring 2023

CSE 30341 is the one of the core classes in the Computer Science and Engineering program at the University of Notre Dame. This course introduces many different aspects of modern operating systems. Topics include process structure and synchronization, interprocess communication, concurrency and parallelism with threads, virtual memory, resource allocation, file systems, security, and distributed systems.
